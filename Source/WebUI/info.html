Commands

  • ⎋ — show/hide system inspector [esc]
  • ⌘ I — show/hide inspector
  • ⌘ A — select all

Clicks

  • click on object — select
  • double click on group — open group
  • double click on other object — select and toggle inspector

Drag

  • drag from output to input to make connection
  • drag from output to widget to set widget source

Not yet implemented

  • ⌫ — delete selected objects
  • ⌘ C — copy selected objects
  • ⌘ X — cut selected objects
  • ⌘ V — paste (and change names if necessary)
  • ⌘ D — duplicate selected objects
  • ⌘ S — save
  • ⌘ O — open
  • ⌘ E — toggle edit mode
  • ⌘ R — start realtime mode
  • ⌘ P — start play mode
  • ⌘ . — pause (or stop if already in pause mode)
  • ⌘ Q — quit ikaros
