# And

## Description

Computes the element-wise logical and of two inputs. Non-zero values are interpreted as true and
zero values as false, and the output is binary with values of 0 or 1.

It receives INPUT1 and INPUT2 and produces OUTPUT. A typical use case is to gate an activity map
with a mask so that only elements that are active in both inputs remain on.
