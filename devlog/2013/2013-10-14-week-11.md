## Narrative

The week is a hardening pass on low-level data handling. XML moved from a larger fixed buffer toward automatic buffer growth, and parsing became stricter through duplicate-attribute checks and entity handling. These changes make configuration loading more robust for larger and messier input files.

The image path was cleaned at the same time. `float_to_byte()` size handling changed, JPEG creation was fixed, and face-detector warnings were removed. WebUI profiling was removed, which simplified the browser-side runtime after earlier profiling experiments.

The week also began a concentrated vision-filter expansion. `IntegralImage`, `GaussianFilter`, and `DoG` entered the repository, setting up the larger filter suite completed the following week.
