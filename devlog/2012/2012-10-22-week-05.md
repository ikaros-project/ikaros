## Week in Brief

This week expanded the vision stack and tightened the build system. Work recorded under `ikaros-project` added and completed `SaliencePoints`, adjusted WebUI and validation details, merged and then removed MarkerTracker work, added viewers for image and salience tools, and introduced the Kinect module. Birger and Birger Johansson updated QTGrab handling, added C++11 and CMake find-module support, improved compiler detection, and added OS X TurboJPEG discovery.

## Highlights

- Added and completed the `SaliencePoints` module, then moved it into `Modules/VisionModules`.
- Added viewers for `InputJpeg`, `MarkerTracker`, and `SaliencePoints`.
- Updated MarkerTracker documentation and parameters, then removed the module from the tree.
- Added WebUI cleanup for temporary files, favicon MIME type correction, validation script support, and edge-detector path fixes.
- Updated QTGrab source handling and removed binary files.
- Added C++11 flags and CMake find-module paths.

## Development Notes

The week is centered on vision modules and build portability. `SaliencePoints` was introduced, completed, and moved into the vision-module hierarchy, making it part of the organized module tree rather than a side branch. Viewers were added for image input, MarkerTracker, and salience output, connecting the module work back to WebUI inspection.

The MarkerTracker line moved quickly. It was merged, documented, parameterized, and then deleted, leaving the history as an integration attempt that did not remain in the active tree. Around that work, the WebUI received practical cleanup: temporary-file removal, favicon MIME correction, validation support, path fixes, and backward-compatible image-object updates.

Birger's build work made the repository more portable. QTGrab source naming was corrected, binary files were removed, C++11 flags and find-module paths were added, GNU compiler detection was investigated, and TurboJPEG discovery was added for OS X. The week ended with the first Kinect module commit and follow-up spelling/configuration fixes.
