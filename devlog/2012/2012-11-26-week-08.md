## Development Notes

The week is a targeted cleanup of YARP integration. `YARPPort` gained a `strict` parameter, giving the module a clearer configuration option for receive behavior. The new CMake find file moved YARP discovery into the same dependency-detection pattern used for other optional libraries.

The capitalization update in the CMake script is part of the same tightening. The repository's build files were still settling into a consistent style after the October CMake expansion, and this week made YARP a more regular member of that build system.
